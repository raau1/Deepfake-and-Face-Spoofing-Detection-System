# Datasets

This directory holds the four evaluation datasets the project is trained and
tested against. The clips themselves are **not redistributed in this
repository** because each dataset is released under an academic license that
prohibits redistribution. Reproducing any of the training or evaluation runs
requires obtaining the originals from the official sources below.

The deployed FastAPI service does **not** require any of these datasets at
runtime; it only needs the trained checkpoints in `outputs/models/`. These
datasets are only needed to retrain a model from scratch or to re-run the
per-dataset evaluations in `scripts/evaluate_model.py`.

`processed/` is generated by the preprocessing scripts; do not download
anything into it.

The standalone `DFD_original sequences/` and `DFD_manipulated_sequences/`
folders hold the Google/Jigsaw DeepFake Detection release. These overlap by
about 1,000 videos with the `DeepFakeDetection/` subfolder inside
FaceForensics++; section 6.10.2 documents the v3 retrain that explicitly
filters those duplicates with `--exclude-video-prefixes`.

## Downloads

All datasets were obtained from Kaggle re-hosts of the original academic
releases. Kaggle requires a free account and accepting each dataset's
Rules tab before download.

| Dataset | Source |
|---|---|
| FaceForensics++ (c23 compressed) | https://www.kaggle.com/datasets/xdxd003/ff-c23 |
| Celeb-DF v2 | https://www.kaggle.com/datasets/reubensuju/celeb-df-v2 |
| DFDC (train sample) | https://www.kaggle.com/competitions/deepfake-detection-challenge/data |
| DeepFakeDetection (DFD) standalone | https://www.kaggle.com/datasets/sanikatiwarekar/deep-fake-detection-dfd-entire-original-dataset |
| WildDeepfake (test subset, 174 clips) | https://www.kaggle.com/datasets/maysuni/wild-deepfake/data?select=test |

The DFD Kaggle dataset unpacks as a single archive containing both
`DFD_original sequences/` and `DFD_manipulated_sequences/`. The 1,000 fake
videos already shipped inside `FaceForensics++/DeepFakeDetection/` are a
subset of the 3,068 fakes in the standalone DFD release, so the two sources
overlap. The v3 retrain in section 6.10.2 explicitly filters this overlap
with `--exclude-video-prefixes` to prevent the same videos appearing twice.

The WildDeepfake link is the test-subset version only (matching the 174
evaluation samples referenced throughout section 6.10); the full
WildDeepfake training split was not used.

The author already had the c23-compressed FaceForensics++ subset and the
first two DFDC train partitions downloaded for the project; only those two
DFDC parts (used in section 6.10.6) are referenced anywhere in the
dissertation.

## Preprocessing

Once the raw clips are in place, regenerate the contents of `processed/` with:

```
python scripts/preprocess_ff_all_types.py
python scripts/preprocess_celebdf.py
python scripts/preprocess_dfd.py
```

Each script reads dataset paths from `config/config.yaml`, runs MTCNN face
detection plus MediaPipe FaceMesh alignment, and saves 32 frames per video
as 299x299 aligned crops under `data/processed/`.

End-to-end preprocessing of FaceForensics++ All-Types takes about 13 hours
on a single RTX 4070; the other datasets are smaller in proportion.
